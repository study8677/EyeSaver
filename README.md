### Option 1: Download Executable (Recommended)

### Option 2: Run from Source
1. Clone the repository:
   ```bash
   git clone https://github.com/study8677/EyeSaver.git
   cd EyeSaver
   ```
2. Install dependencies using `uv` (or pip):
   ```bash
   uv sync
   ```
3. Run the application:
   ```bash
   uv run src/main.py
   ```
